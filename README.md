Anexados aqui estão: um relatório detalhando e explicando o código, os códigos em arquivos do tipo C e os aplicativops executáveis, sendo o SC, a versão sem comentários no código, e a CC, a versão com comentários, e o dataset utilizado, no formato CSV.

Compilação e Execução
Pré-requisitos: GCC instalado (MinGW no Windows, GCC nativo no Linux/Mac).
Windows (executável pronto):
Coloque protótipo002CC.exe e players_22.csv na mesma pasta e execute diretamente com duplo clique ou via terminal:
protótipo002CC.exe
Linux / Mac (compilar do zero):
gcc protótipo002CC.cpp -o prototipo -lm
./prototipo
O arquivo players_22.csv deve estar na mesma pasta que o executável em qualquer sistema.

Os arquivos CC contêm comentários pedagógicos no código. Os arquivos SC são funcionalmente idênticos, sem comentários.
